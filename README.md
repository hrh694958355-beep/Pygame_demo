# Pygame_demo: Space Shooter with DQN Reinforcement Learning

## 项目简介
本项目以经典「飞机大战」游戏为载体，基于 Pygame 搭建强化学习环境，结合深度Q网络（DQN）实现智能体（Agent）训练。智能体通过与环境交互试错，学习「躲避敌方飞机+精准射击」的最优策略，验证了 DQN 算法在高维视觉状态决策任务中的有效性。

## 核心特性
- ✨ 完整的游戏环境：包含玩家控制、敌方AI、子弹碰撞、奖励机制等核心逻辑；
- 🧠 DQN 算法优化：整合经验回放、目标网络、ε-贪心策略，保证训练稳定性；
- ⚡ 训练加速：支持「无渲染训练」（速度提升2~8倍）和「可视化训练」切换；
- 📈 性能评估：记录训练过程得分/损失变化，可直观分析模型学习效果。

核心算法

DQN 核心公式

$$Q(s, a) \leftarrow Q(s, a) + \alpha \left[ R + \gamma \max_{a'} Q(s', a') - Q(s, a) \right]$$

- $$s, a$$：当前状态/动作

- $$R$$：即时奖励

- $$s'$$：执行动作后的新状态

- $$\alpha$$：学习率，$$\gamma$$：折扣因子

关键优化

1. 经验回放：解决数据相关性问题，随机采样训练；

2. 目标网络：独立计算目标Q值，避免训练震荡；

3. ε-贪心策略：平衡探索（随机动作）与利用（最优动作）。

常见问题

1. 训练速度慢：开启「无渲染训练」，关闭游戏画面渲染；

2. 训练不稳定：检查奖励函数设计，确保正负奖励合理；

3. 推送GitHub失败：改用SSH协议，配置SSH密钥后重试。
